# Docker Shopware

## ENVs
Docker environment params

### Base

#### NO_SW
(default: not set)
Don't install shopware.

### Shopware

#### SW_VERSION
(default: Latest)

### Database

#### DB_HOST
(default: "db")

#### DB_PORT
(default: "3306")

#### DB_USER
(default: NONE)

#### DB_PASSWORD
(default: NONE)

#### DB_NAME
(default: NONE)

### Shop

#### SHOP_LOCALE
(default: "de_DE")

#### SHOP_HOST
(default: "localhost")

#### SHOP_PATH
(default: "/")

#### SHOP_NAME
(default: "Demo shop")

#### SHOP_EMAIL
(default: "your.email@shop.com")

#### SHOP_CURRENCY
(default: "EUR")

### Admin

#### ADMIN_USER
(default: "demo")

#### ADMIN_PASSWORD
(default: "demo")

#### ADMIN_EMAIL
(default: "demo@demo.demo")

#### ADMIN_LOCALE
(default: "de_DE")

#### ADMIN_NAME
(default: "Demo user")
